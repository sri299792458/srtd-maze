# Implementation spec: Spectral Residual Trajectory Diffusion on Ambient 2D Maze

## 0. Goal

Implement and evaluate **Spectral Residual Trajectory Diffusion**, abbreviated **SRTD**, on the Ambient 2D Maze dataset.

The hypothesis:

> RRT trajectories are useful for low-frequency global route information but harmful in high-frequency residual bands. A policy should learn low-frequency route structure from (D_q), suppress high-frequency RRT jitter, and emphasize clean residual events from (D_p), such as turns and obstacle-avoidance corrections.

This combines three ideas:

Sander Dieleman’s diffusion-as-spectral-autoregression observation gives the mapping from diffusion noise level to detectable frequency bands. In his formulation, adding Gaussian noise progressively hides high frequencies first, so diffusion denoising behaves like approximate frequency-space autoregression. ([Sander Dieleman][1])

Hou and Zhang’s spectral residual method gives the innovation operator: compute log amplitude spectrum, subtract a smoothed spectral envelope, preserve phase, and inverse-transform the residual to obtain a saliency map. 

Ambient Diffusion Policy supplies the robotics setting: robot action data exhibits a spectral power law, Diffusion Policies show a global-to-local hierarchy, and suboptimal data can be used at selected diffusion times instead of all times. 

## 1. What Codex should build

Build three new components on top of a Diffusion Policy maze implementation:

1. **Trajectory spectral residual diagnostics**
   Compute trajectory PSDs, spectral envelopes, residual spectra, and time-localized residual saliency maps for GCS and RRT trajectories.

2. **Spectral-residual Ambient sampler**
   Replace Ambient’s scalar (t_{\min}) for all RRT data with a **per-chunk (t_{\min}^{SR})** derived from how much bad high-frequency residual remains visible at each diffusion noise level.

3. **Frequency-masked denoising loss**
   Train on RRT data in frequency space, using low-frequency compatible bands while suppressing high-frequency residual bands that look unlike clean GCS trajectories.

The resulting experiment should compare:

| Name             |                                        Data used | Loss                 | Purpose                         |
| ---------------- | -----------------------------------------------: | -------------------- | ------------------------------- |
| `gcs_only`       |                                       (D_p) only | standard x0 MSE      | smooth but underfit baseline    |
| `rrt_only`       |                                       (D_q) only | standard x0 MSE      | high-coverage, jittery baseline |
| `cotrain`        |                  (D_p + D_q) all diffusion times | standard x0 MSE      | standard mixed-data baseline    |
| `ambient_scalar` |           (D_p) all times, (D_q) only high noise | standard x0 MSE      | Ambient baseline                |
| `sr_tmin`        |  (D_p) all times, (D_q) gated by (t_{\min}^{SR}) | standard x0 MSE      | lowest-risk new method          |
| `sr_freqmask`    |          (D_p) all times, (D_q) frequency-masked | spectral masked loss | main new method                 |
| `sr_full`        | (D_p) saliency-weighted + (D_q) frequency-masked | time + spectral loss | full method                     |

## 4. Spectral residual core

### 4.1 FFT convention

Actions have shape:

```python
A: torch.Tensor  # [B, H, D], D=2
```

Use real FFT along the time axis:

```python
X = torch.fft.rfft(A, dim=1, norm="ortho")  # [B, F, D]
amp = torch.abs(X)
phase = torch.angle(X)
log_amp = torch.log(amp + eps)
```

Default:

```yaml
eps: 1.0e-6
basis: rfft
envelope_kernel_size: 3
```

### 4.2 Local spectral residual

Implement the Hou-Zhang-style residual for each chunk:

[
L(f,d) = \log(|\hat A(f,d)| + \epsilon)
]

[
\bar L(f,d) = \text{AvgPoolFreq}_{k=3}(L)(f,d)
]

[
R(f,d) = L(f,d) - \bar L(f,d)
]

Use reflect padding in frequency before average pooling. Do not smooth across action dimensions.

Function:

```python
def spectral_residual(
    actions: torch.Tensor,
    kernel_size: int = 3,
    eps: float = 1e-6,
) -> dict:
    """
    actions: [B, H, D]
    returns:
      X: complex [B, F, D]
      log_amp: [B, F, D]
      smooth_log_amp: [B, F, D]
      residual: [B, F, D]
      phase: [B, F, D]
      saliency_time: [B, H, D]
      saliency_scalar: [B, H]
    """
```

### 4.3 Time-localized saliency

Preserve phase and inverse-transform:

[
S(t,d)=
\left|
\mathcal{F}^{-1}
\left[
\exp(R(f,d))e^{i\phi(f,d)}
\right]
\right|^2
]

Then aggregate over dimensions:

[
S(t)=\sum_d S(t,d)
]

Normalize per chunk:

```python
S = S / (S.mean(dim=1, keepdim=True) + eps)
```

For diagnostics, mark proto-events with:

```python
event_mask = S > 3.0 * S.mean(dim=1, keepdim=True)
```

That threshold mirrors Hou and Zhang’s simple proto-object thresholding rule (S(x) > 3E[S(x)]).

```python
class CleanSpectralStats:

    clean_psd_mean: torch.Tensor         # [F, D]
```

Fitting procedure:

```python
for all clean training chunks A_p:
    X = rfft(A_p)
    log_amp = log(abs(X) + eps)
    R = log_amp - avgpool_freq(log_amp, k=3)

clean_log_amp_median = median(log_amp over chunks)
clean_log_amp_q90 = quantile(log_amp, 0.90 over chunks)
clean_residual_q95 = quantile(R, 0.95 over chunks)
clean_psd_mean = mean(psd over chunks)
```

## 6. Noise-to-frequency visibility map

Use a VP diffusion process:

[
A_t = \sqrt{1-\sigma_t^2}A_0 + \sigma_t \epsilon
]

Estimate target SNR per frequency using clean (D_p) PSD:

[
\text{SNR}(f,d,t) =
\frac{(1-\sigma_t^2)P_p(f,d)}
{\sigma_t^2 + \epsilon}
]

Define soft visibility:

[
V(f,d,t)=
\sigma_{\text{logistic}}
\left(
\frac{
\log(\text{SNR}(f,d,t)+\epsilon)-\log(\tau_{\text{snr}})
}{
T_{\text{vis}}
}
\right)
]

Default config:

```yaml
snr_tau: 1.0
visibility_temperature: 0.5
min_visible_weight: 0.02
```

Function:

```python
def visibility_mask(
    clean_psd_mean: torch.Tensor,  # [F, D]
    sigma: torch.Tensor,           # [B] or scalar
    tau: float = 1.0,
    temperature: float = 0.5,
) -> torch.Tensor:
    """
    returns V: [B, F, D]
    """
```

Unit-test requirement: as (\sigma_t) increases, the highest visible frequency must be non-increasing.

## 7. Spectral-residual (t_{\min}^{SR}) annotation

This is the first new method and should be implemented before the full loss.

For each chunk, compute “bad residual” relative to the clean residual distribution:

[
B_i(f,d)=\max(0, R_i(f,d)-Q^{p}_{0.95}(R(f,d))-\delta)
]

Default:

```yaml
bad_residual_margin: 0.0
global_band_cutoff_norm: 0.20
```

Do not penalize very low frequencies. They encode route/global plan and are the part of RRT we want to keep.

```python
f_norm = freq_idx / (F - 1)
high_or_mid_band = f_norm > global_band_cutoff_norm
B = B * high_or_mid_band
```

For every candidate diffusion time (t \in {0,\dots,T-1}), compute the residual still visible:

[
\text{bad_visible}*i(t)=
\frac{
\sum*{f,d} B_i(f,d)V(f,d,t)
}{
\sum_{f,d} V(f,d,t)+\epsilon
}
]

Compute the same score for clean chunks and store:

[
\eta_p(t) = Q_{0.95}\left(\text{bad_visible}_p(t)\right)
]

Then for every RRT chunk (i):

[
t_{\min,i}^{SR}
===============

\min{t: \text{bad_visible}_i(t)\leq \eta_p(t)}
]

If no such (t) exists, set `tmin = T`, meaning the chunk is never used.

Function:

```python
def annotate_sr_tmin(
    chunks: list[MazeChunk],
    clean_stats: CleanSpectralStats,
    schedule: VPSchedule,
    cfg: SpectralConfig,
) -> np.ndarray:
    """
    returns sr_tmin_idx: int array [num_chunks]
    """
```

Save:

```text

```

## 9. Losses

### 9.1 Standard x0 denoising loss

Use x0-prediction:

```python
sigma = schedule[t_idx]
A_t = sqrt(1 - sigma**2) * A0 + sigma * eps
A0_pred = model(A_t, obs, t_idx)
loss = mse(A0_pred, A0)
```

### 9.2 Frequency-domain MSE

Use full complex FFT or rFFT. For simplicity with one-sided rFFT, include real-frequency weights carefully. Easier implementation: use full FFT for the loss.

```python
def spectral_mse(
    pred: torch.Tensor,       # [B, H, D]
    target: torch.Tensor,     # [B, H, D]
    weights: torch.Tensor,    # [B, H, D] full FFT weights
) -> torch.Tensor:
    E = torch.fft.fft(pred - target, dim=1, norm="ortho")
    loss_fd = (E.real.square() + E.imag.square()) * weights
    return loss_fd.sum() / (weights.sum() + 1e-8)
```

Unit test: when `weights = ones`, this should match time-domain MSE up to numerical tolerance by Parseval.

### 9.3 Clean saliency-weighted loss for (D_p)

For clean chunks, emphasize residual-salient timesteps:

```python
saliency = precomputed_or_online_saliency(A0)  # [B, H]
saliency = saliency / (saliency.mean(dim=1, keepdim=True) + eps)
time_weight = 1.0 + beta_clean_saliency * clamp(saliency, max=saliency_clip)
loss_p = weighted_time_mse(A0_pred, A0, time_weight)
```

Defaults:

```yaml
beta_clean_saliency: 0.25
saliency_clip: 5.0
```

This is intentionally mild. The clean dataset is tiny, so do not overfit every corner.

### 9.4 RRT frequency-masked loss for (D_q)

For each RRT chunk:

[
C_i(f,d)=
\text{sigmoid}
\left(
\frac{
Q^{p}_{0.95}(R(f,d))+\delta-R_i(f,d)
}{
T_C
}
\right)
]

This compatibility is near 1 where the RRT residual looks clean-like and near 0 where it has abnormal high-frequency residual.

Always allow low-frequency bands:

```python
C[:, f_norm <= 0.20, :] = 1.0
```

Then combine with diffusion-time visibility:

[
W_i(f,d,t)=V(f,d,t)C_i(f,d)
]

Clamp low-frequency floor:

```python
W[:, f_norm <= 0.20, :] = torch.clamp(W[:, f_norm <= 0.20, :], min=0.25)
W = torch.clamp(W, min=0.0, max=1.0)
```

Loss:

```python
loss_q = spectral_mse(A0_pred, A0, full_fft_symmetric(W))
```

Default:

```yaml
compat_temperature: 0.25
low_freq_floor: 0.25
q_loss_weight: 1.0
```

### 9.5 Full SRTD batch loss

```python
if source == "p":
    loss = loss_p_time_saliency + lambda_p_freq * spectral_mse_all_bands
elif source == "q":
    loss = q_loss_weight * loss_q_freq_masked
```

Defaults:

```yaml

```

For the first implementation, do not combine this with Ambient loss. Use standard denoising MSE so failures are interpretable.

## 13. Acceptance criteria

The implementation is successful if all of these pass:

### Diagnostic gates

1. `mean_high_freq_residual(RRT) > mean_high_freq_residual(GCS)`.
2. A simple classifier using only high-frequency residual energy gets AUROC ≥ 0.75 for GCS vs RRT chunks.
3. Visibility mask is monotonic: as noise increases, the maximum visible frequency does not increase.
4. Residual saliency has positive correlation with squared acceleration on RRT rollouts.

### Policy gates

Using 1,000 shared eval trials per policy:

1. `sr_tmin` should match `ambient_scalar` success within 1–2 percentage points.
2. `sr_tmin` should be smoother than `cotrain`.
3. `sr_freqmask` should have lower generated high-frequency residual energy than `cotrain`.
4. `sr_full` should either:

   * improve smoothness over `ambient_scalar` at statistically similar success, or
   * improve success over `ambient_scalar` at statistically similar smoothness.

A good first target is:

```text
success_rate(sr_full) >= success_rate(ambient_scalar) - 0.01
smoothness(sr_full) <= smoothness(ambient_scalar) * 0.95
hf_residual_energy(sr_full) <= hf_residual_energy(cotrain) * 0.80
```

Do not tune against the eval set. Use validation rollouts or a held-out start-goal set.

## 15. Main implementation pseudocode

### Training loop

```python
for step in range(total_steps):
    batch = sampler.sample(batch_size)

    A0 = batch.actions              # [B, H, 2]
    obs = batch.obs                 # [B, obs_dim]
    source = batch.source_id        # [B]
    t_idx = batch.t_idx             # [B]

    sigma = schedule.sigma[t_idx]   # [B]
    eps = torch.randn_like(A0)
    At = sqrt(1 - sigma**2)[:, None, None] * A0 + sigma[:, None, None] * eps

    A0_pred = model(At, obs, t_idx)

    if cfg.method in ["gcs_only", "rrt_only", "cotrain", "ambient_scalar", "sr_tmin"]:
        loss = mse(A0_pred, A0)

    elif cfg.method == "sr_freqmask":
        loss = sr_freqmask_loss(A0_pred, A0, source, t_idx, clean_stats, cfg)

    elif cfg.method == "sr_full":
        loss = sr_full_loss(A0_pred, A0, source, t_idx, clean_stats, cfg)

    optimizer.zero_grad()
    loss.backward()
    clip_grad_norm_(model.parameters(), cfg.training.grad_clip_norm)
    optimizer.step()
    ema.update(model)
```

### `sr_full_loss`

```python
def sr_full_loss(pred, target, source, t_idx, clean_stats, cfg):
    losses = []

    p_mask = source == 0
    q_mask = source == 1

    if p_mask.any():
        sal = spectral_residual(target[p_mask])["saliency_scalar"]  # [Bp, H]
        sal = sal / (sal.mean(dim=1, keepdim=True) + cfg.eps)
        sal = sal.clamp(max=cfg.spectral.saliency_clip)
        time_w = 1.0 + cfg.spectral.beta_clean_saliency * sal
        loss_p_time = weighted_time_mse(pred[p_mask], target[p_mask], time_w)

        Wp = torch.ones_like_full_fft_weights(target[p_mask])
        loss_p_freq = spectral_mse(pred[p_mask], target[p_mask], Wp)

        losses.append(loss_p_time + cfg.spectral.lambda_p_freq * loss_p_freq)

    if q_mask.any():
        Wq_rfft = compute_q_frequency_weights(
            target[q_mask],
            t_idx[q_mask],
            clean_stats,
            cfg,
        )
        Wq_fft = make_symmetric_full_fft_weights(Wq_rfft, H=target.shape[1])
        loss_q = spectral_mse(pred[q_mask], target[q_mask], Wq_fft)
        losses.append(cfg.spectral.q_loss_weight * loss_q)

    return sum(losses) / len(losses)
```

## 16. Practical failure modes and fixes

If `sr_tmin` uses almost no RRT chunks, loosen the threshold:

```yaml
clean_bad_visible_quantile: 0.99
bad_residual_margin: 0.1
global_band_cutoff_norm: 0.30
```

If `sr_freqmask` underfits global planning, raise the low-frequency floor:

```yaml
low_freq_floor: 0.5
global_band_cutoff_norm: 0.25
```

If SRTD becomes smooth but fails more, the mask is suppressing mid-frequency turns. Inspect `residual_saliency_rollouts.png` and reduce `compat_temperature` aggressiveness or allow mid bands.

If H=16 is too short for stable spectral residuals, keep policy horizon at 16 for comparability but compute annotations from full 100-step episode windows. The training loss can remain chunk-level.

If generated paths cut corners into obstacles, add a collision-margin metric and report it separately; do not hide this under smoothness.
